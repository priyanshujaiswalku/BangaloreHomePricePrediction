# 🏡 Bangalore Home Price Prediction

A full-stack Machine Learning web application that predicts real estate prices in Bangalore, India based on features like square footage, number of bedrooms (BHK), bathrooms, and location.

---

## 🔗 Repository Link
https://github.com/priyanshujaiswalku/BangaloreHomePricePrediction

---

## 📁 Project Structure

This project is divided into three main parts:

- **model/**  
  Contains Jupyter Notebooks for:
  - Data Cleaning  
  - Exploratory Data Analysis (EDA)  
  - Model Training  

- **server/**  
  Flask backend:
  - Loads trained model  
  - Handles API requests  
  - Returns predictions  

- **client/**  
  Frontend built with:
  - HTML  
  - CSS  
  - JavaScript  

---

## 🛠️ Tech Stack

- Machine Learning: Python, Pandas, NumPy, Scikit-learn  
- Backend: Flask  
- Frontend: HTML, CSS, JavaScript
